This is a sample HTML document for testing the enhanced ingestion pipeline. It demonstrates how HTML content is processed and extracted.

HTML Processing Features

The HTML loader extracts visible text content while preserving structure. It removes script and style elements to focus on meaningful content.

Key features include:

  • Title extraction from <title> tags
  • Text extraction from body content
  • Paragraph-based chunking
  • Metadata preservation

Technical Implementation

The HTML processing uses BeautifulSoup for parsing and text extraction. It handles various HTML structures and preserves document hierarchy.
